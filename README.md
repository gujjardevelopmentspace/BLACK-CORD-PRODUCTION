# BLACK-CORD-PRODUCTION
A responsive website built with React, Vite, and TailwindCSS.
It includes modern UI, smooth animations, and sections for services, portfolio, and contact.

✨ Features

🎨 Modern design (black, white, red theme)

📱 Responsive layout for all devices

🎥 Smooth animations with GSAP

📂 Services & Portfolio showcase

📩 Contact form integration

🛠 Tech Stack

⚛️ React

⚡ Vite

🎨 TailwindCSS

🧩 shadcn/ui + lucide-react

🎞 GSAP

🚀 Getting Started

Clone the repo:

git clone https://github.com/yourusername/main-website.git
cd main-website


Install dependencies:

npm install


Run locally:

npm run dev
